# Nodepop

To start the application use:

```sh
npm install
```

In production:

```sh
npm start
```

In development:

```sh
npm run dev
```


## Inicializar la BD

Para inicializar la BD al estado inicial, se puede usar el comando:

```sh
npm run initdb
```

* ATENCIÓN * Esto borrará todos los datos de la BD y cargará el estado inicial.
